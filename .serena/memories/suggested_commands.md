# Project Purpose
OBGP platform providing OSC management, transparency, blog, experiences, and a public landing page.

# Tech Stack
- Next.js 16.2.4 (React 19)
- TypeScript
- Supabase for backend / database / auth
- CSS for styling

# Available Commands
- `npm run dev`: start development server
- `npm run build`: build production app
- `npm run start`: start production app
- `npm run lint`: run linter
- `npm run test`: run vitest

# Conventions
- Uses standard Next.js App Router structure in `src/app`.
- Reusable UI components in `src/components/ui`.
- Shared Admin UI in `src/app/gestao/dashboard/_shared/AdminUI.tsx`.
